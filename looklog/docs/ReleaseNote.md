# 版本记录

### v1.1.0 - 2024-11-26

**🥳新功能：**
  * 在衣柜视图顶部添加分类筛选按钮，点击可切换不同类别的衣物列表。
  * 添加iCloud备份功能，支持在不同设备间同步数据。

**💅改进：**
  * 优化了图片存储大小，减少内存占用。



### v1.0.0 - 2024-11-25


**🥳新功能：**
* **穿搭记录**
  * 记录每日穿搭，自动去除背景，但可以在详情页显示原图。

* **衣物管理**
  * 添加、编辑和删除衣物信息。
  * 通过拍照或从相册中选择图片添加衣物。

* **分类浏览**
  * 根据类别查看衣物，包括上衣、外套等。

* **穿着次数记录**
  * 记录每件衣物的穿着次数，帮助用户了解衣物的使用频率。
  * 提供“一键+1”按钮，快速更新穿着次数。

* **价格记录**
  * 为每件衣物添加购买价格，方便统计衣物花费。

* **衣物详情**
  * 查看衣物的详细信息，包括图片、类别、价格和穿着次数。
  * 支持编辑衣物信息，修改错误或更新内容。


**附加说明**

* **数据安全**：所有数据均保存在本地设备中，保障您的隐私安全。
